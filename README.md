# FUTURE_PE_03 – Prompt Evaluation System

## Overview
This project demonstrates a simple prompt evaluation framework.

It compares different prompts and evaluates how prompt improvements affect AI-generated outputs.

## Objective
The goal of this project is to understand how prompt design impacts AI responses.
